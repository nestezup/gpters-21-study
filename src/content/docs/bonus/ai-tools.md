# AI 코딩 도구 — Claude Code와 Cursor

## 1. AI 코딩 도구가 뭔가요?

AI 코딩 도구는 여러분이 자연어로 지시하면 코드를 대신 작성해 주는 비서입니다. 전통적인 코딩은 문법을 외워서 직접 타이핑하는 방식이었지만, AI 도구를 쓰면 "토마토 파스타 만들어줘"처럼 원하는 결과만 말하면 됩니다.

단, AI가 항상 완벽하지는 않으므로 결과를 확인하고 되돌릴 수 있는 능력(Git 등)이 중요합니다.

---

## 2. Claude Code — 터미널에서 동작하는 AI

Anthropic이 만든 터미널 기반 AI 도구입니다. 별도 편집기 없이 명령창에서 바로 파일 생성, 수정, 삭제를 수행합니다.

### STEP 1: Node.js 설치 확인

Claude Code는 Node.js가 필요합니다. 터미널에서 확인합니다.

```bash
node --version
# v20.x.x 같은 숫자가 나오면 OK
# 안 나오면 2주차 fnm 가이드를 참고하여 Node.js를 먼저 설치하세요
```

### STEP 2: Claude Code 설치

```bash
npm install -g @anthropic-ai/claude-code
```

설치 확인:

```bash
claude --version
```

### STEP 3: 프로젝트 폴더에서 실행

```bash
mkdir my-project
cd my-project
claude
```

### STEP 4: 인증

처음 실행하면 브라우저가 열립니다. Anthropic 계정으로 로그인하고 승인 버튼을 누르면 터미널로 돌아와 대화를 시작할 수 있습니다.

### 사용 예시

- `"index.html 파일을 만들어줘. 안녕하세요 세계라고 표시하는 웹페이지로."` — 파일 자동 생성
- `"이 프로젝트에 있는 파일 목록을 보여줘"` — 프로젝트 구조 파악
- `"hello.js 파일의 내용을 일본어로 바꿔줘"` — 기존 파일 수정

Claude Code는 단순히 텍스트를 출력하는 것이 아니라, 실제로 파일을 만들고 명령어를 대신 실행해 주는 도구입니다.

### 비용 안내

Claude Code는 두 가지 방식으로 사용할 수 있습니다. 첫째, Anthropic의 Claude 구독(Pro $20/월, Max $100/월 또는 $200/월)에 가입하면 구독 범위 내에서 사용할 수 있습니다. 둘째, API 계정을 만들어 사용한 만큼 비용을 지불하는 방식도 있습니다. 입문자에게는 Pro 구독($20/월)으로 시작하는 것을 권장합니다.

## 3. Cursor — AI가 내장된 코드 에디터

VS Code 기반의 코드 편집기로, 화면에서 코드를 보면서 AI와 대화할 수 있습니다.

### STEP 1: 설치 파일 다운로드

1. https://cursor.com 에 접속합니다.
2. 운영체제에 맞는 파일을 받습니다 (Windows: `.exe`, Mac: `.dmg`).

### STEP 2: 설치 및 실행

내려받은 파일을 실행하여 안내에 따라 설치를 완료합니다. 기존 VS Code 설정이 있으면 자동으로 가져올 수 있습니다.

### STEP 3: 핵심 단축키

| 단축키 | Mac | Windows | 기능 |
| :--- | :--- | :--- | :--- |
| 인라인 편집 | `Cmd+K` | `Ctrl+K` | 선택한 코드 영역에 수정 요청 |
| 채팅 창 | `Cmd+L` | `Ctrl+L` | 프로젝트 전체에 대해 질문 |
| 자동 완성 | `Tab` | `Tab` | AI가 제안한 코드 수락 |

### 사용 예시

- 코드 영역 선택 → `Cmd+K` → `"에러 처리 코드를 추가해줘"` — 인라인 수정
- 채팅 창 → `"이 프로젝트가 어떻게 동작하는지 설명해줘"` — 전체 구조 파악
- 코드 작성 중 AI가 다음 내용을 흐릿하게 제안 → `Tab`으로 수락

### 비용 안내

Cursor는 무료 Hobby 플랜과 유료 Pro 플랜($16~20/월)으로 나뉩니다. 무료 플랜에서도 제한된 횟수의 AI 요청을 사용할 수 있어 입문 단계에서는 충분합니다.

## 4. Claude Code vs Cursor — 어떤 걸 쓸까?

| 항목 | Claude Code | Cursor |
| :--- | :--- | :--- |
| 동작 방식 | 터미널 기반 (명령어 입력 방식) | 코드 편집기 기반 (시각적 방식) |
| 설치 방법 | npm을 통한 명령어 설치 | 전용 설치 파일 다운로드 |
| 시각적 편집 | 텍스트 대화로만 진행 | 화면에서 직접 코드를 보며 수정 |
| 파일 관리 | AI가 직접 생성하고 관리 | 사용자가 직접 확인하며 수정 가능 |
| 추천 상황 | 프로젝트 초기 설정이나 빠른 실행 시 | 코드의 세부 내용을 보며 고쳐나갈 때 |

두 도구를 모두 설치해 두는 것을 권장합니다. 프로젝트 초기 설정이나 대량 작업에는 Claude Code를, 코드를 세밀하게 다듬을 때는 Cursor를 사용하면 효과적입니다.

## 5. 자주 만나는 문제

| 상황 | 원인 | 해결 |
|------|------|------|
| `npm: command not found` | Node.js 미설치 | Node.js 먼저 설치 후 재시도 |
| 인증 화면이 안 열림 | 브라우저 팝업 차단 | 팝업 차단 해제 후 재시도 |
| Cursor 무료 한도 초과 | 월간 사용량 소진 | 다음 달까지 대기 또는 유료 전환 |
| AI 응답이 느림 | 네트워크 문제 또는 서버 부하 | 인터넷 연결 확인, 잠시 후 재시도 |
| AI가 작성한 코드가 작동 안 함 | AI의 실수 (빈번하게 발생) | `"이전 상태로 되돌려줘"` 요청 또는 Git으로 복구 |

## 6. 도구 삭제하기

**Claude Code:**

```bash
npm uninstall -g @anthropic-ai/claude-code
```

**Cursor:**
- Windows: 제어판 → 프로그램 추가/제거
- Mac: 응용 프로그램 폴더에서 Cursor를 휴지통으로 이동

---

## 7. 정리

Claude Code는 터미널에서 빠르게 명령을 수행하는 도구이고, Cursor는 코드를 시각적으로 확인하며 수정하는 도구입니다. 상황에 맞게 골라 쓰면서, AI의 결과물을 검증하고 되돌릴 수 있는 능력을 함께 키워 나가세요.
