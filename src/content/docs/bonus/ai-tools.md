# AI 코딩 도구 — Claude Code와 Cursor

안녕하세요. 코딩의 세계에 발을 들이신 여러분을 환영합니다. 예전에는 컴퓨터와 대화하기 위해 아주 복잡하고 낯선 언어를 완벽하게 외워야만 했습니다. 하지만 이제는 인공지능 기술의 발전 덕분에 우리가 평소에 사용하는 말로도 훌륭한 프로그램을 만들 수 있는 시대가 되었습니다. 이번 장에서는 바이브코딩의 핵심 동반자인 두 가지 도구, Claude Code와 Cursor에 대해 자세히 알아보겠습니다.

## 1. AI 코딩 도구가 뭔가요?

AI 코딩 도구는 여러분의 곁에서 코드를 대신 작성해 주는 숙련된 비서라고 생각하시면 이해가 빠릅니다. 비유를 들어보겠습니다. 여러분이 고급 레스토랑의 주방장이라고 가정해 봅시다. 예전의 전통적인 코딩 방식은 주방장이 재료 손질부터 불 조절, 설거지까지 모든 과정을 직접 언어를 공부해서 수행하는 것과 같았습니다. 레시피를 하나라도 틀리면 요리가 완성되지 않았고, 주방장은 밤을 새워가며 문법 책을 뒤져야 했습니다.

하지만 바이브코딩 환경에서의 AI 코딩 도구는 다릅니다. 여러분은 주방장으로서 "이번에는 신선한 토마토 소스를 곁들인 파스타를 만들어줘"라고 말만 하면 됩니다. 그러면 곁에 있는 인공지능 비서가 즉시 재료를 준비하고 적절한 온도로 조리하여 결과물을 내놓습니다. 여러분은 비서가 만든 요리가 의도대로 잘 나왔는지 확인하고, 소스가 조금 더 맵게 되었으면 좋겠다고 추가 요청만 하면 됩니다.

물론 인공지능 비서가 항상 완벽한 것은 아닙니다. 가끔은 설탕 대신 소금을 넣는 실수를 할 때도 있습니다. 그렇기 때문에 요리를 시작하기 전의 주방 상태를 잘 기록해 두고, 비서가 실수를 했을 때 언제든지 이전 상태로 되돌릴 수 있는 능력이 매우 중요합니다. 이것이 바로 우리가 환경 세팅과 버전 관리를 중요하게 생각하는 이유입니다.

앞으로 살펴볼 두 가지 도구인 Claude Code와 Cursor는 이 인공지능 비서를 만나는 통로가 서로 다릅니다. 하나는 검은색 화면의 터미널에서 글자로 대화하는 방식이고, 다른 하나는 우리가 익숙한 문서 편집기 안에서 코드를 직접 보며 대화하는 방식입니다.

## 2. Claude Code — 터미널에서 동작하는 AI

### Claude Code란 무엇인가요?
Claude Code는 Anthropic이라는 인공지능 연구소에서 만든 도구입니다. 이 도구의 가장 큰 특징은 터미널이라고 불리는 검은색 명령창 안에서 직접 동작한다는 점입니다. 별도의 화려한 화면은 없지만, 여러분이 내리는 명령을 가장 직접적으로 알아듣고 실행하는 능력을 갖추고 있습니다. 마치 무전기를 통해 현장에 있는 요원에게 지시를 내리는 것과 비슷합니다.

### 설치하기
Claude Code를 사용하기 위해서는 먼저 컴퓨터에 Node.js라는 프로그램이 설치되어 있어야 합니다. 이 프로그램은 Claude Code가 동작할 수 있는 기초 토대가 됩니다. 기초 토대가 준비되었다면 터미널을 열고 아래의 명령어를 입력하여 설치를 진행합니다.

```bash
npm install -g @anthropic-ai/claude-code
```

설치가 완료되었는지 확인하려면 다음 명령어를 입력해 봅니다. 설치된 버전 정보가 나타난다면 성공입니다.

```bash
claude --version
```

### 처음 실행하기
이제 여러분의 첫 프로젝트 폴더를 만들고 Claude Code를 불러보겠습니다. 터미널에서 다음 과정을 따라가 보세요.

```bash
mkdir my-project
cd my-project
claude
```

처음 실행하면 인증 절차가 진행됩니다. 화면에 나타나는 안내에 따라 브라우저가 열리면 Anthropic 계정으로 로그인을 하고 승인 버튼을 누르면 됩니다. 인증이 완료되면 다시 터미널로 돌아와 인공지능 비서와 대화를 시작할 수 있습니다.

### 사용 예시
Claude Code에게 다음과 같은 요청을 직접 해볼 수 있습니다.

첫 번째 요청: "index.html 파일을 만들어줘. 안녕하세요 세계라고 표시하는 웹페이지로."
이 요청을 받으면 Claude Code는 즉시 폴더 안에 파일을 생성하고 그 안에 필요한 코드를 작성합니다. 여러분이 직접 코드를 입력할 필요가 없습니다.

두 번째 요청: "이 프로젝트에 있는 파일 목록을 보여줘"
프로젝트의 구조를 파악하고 싶을 때 유용합니다. 인공지능은 현재 어떤 파일들이 있는지 정리해서 알려줄 것입니다.

세 번째 요청: "hello.js 파일의 내용을 일본어로 바꿔줘"
이미 존재하는 파일의 내용을 수정하는 것도 가능합니다. 특정 부분만 고치거나 전체적인 내용을 번역하는 일도 자연스럽게 수행합니다.

Claude Code는 단순히 글자만 쓰는 것이 아니라, 실제로 파일을 만들고 삭제하며 컴퓨터의 명령어를 대신 실행해 주는 실천적인 비서입니다.

### 비용 안내
Claude Code를 사용하려면 Anthropic의 API 계정이 필요합니다. 사용한 만큼 비용이 발생하는 구조이며, 인공지능이 읽고 쓰는 데이터의 양에 따라 계산됩니다. 처음 가입 시 일정 금액의 무료 혜택을 제공하는 경우가 있으니 공식 홈페이지의 안내를 확인해 보시기 바랍니다.

## 3. Cursor — AI가 내장된 코드 에디터

### Cursor란 무엇인가요?
Cursor는 우리가 흔히 사용하는 메모장이나 워드 프로세서와 비슷한 형태의 코드 편집기입니다. 전 세계 개발자들이 가장 많이 사용하는 Visual Studio Code라는 프로그램을 기반으로 만들어졌기 때문에 매우 친숙한 화면 구성을 가지고 있습니다. 코드를 눈으로 직접 확인하면서 필요한 부분을 마우스로 선택하고 수정 요청을 보낼 수 있다는 점이 매력적입니다.

### 설치하기
Cursor는 전용 홈페이지에서 내려받아 설치할 수 있습니다.
1. https://cursor.com 에 접속합니다.
2. 사용 중인 컴퓨터의 운영체제에 맞는 설치 파일을 선택합니다. 윈도우 사용자라면 .exe 파일을, 맥 사용자라면 .dmg 파일을 받으면 됩니다.
3. 내려받은 파일을 실행하여 안내에 따라 설치를 완료합니다.

### 처음 실행하기
설치 후 Cursor를 실행하면 기존에 Visual Studio Code를 사용하던 분들은 설정이나 확장 프로그램을 그대로 가져올 수 있다는 안내가 나옵니다. 처음 사용하시는 분들도 직관적인 화면 구성 덕분에 큰 어려움 없이 적응하실 수 있습니다.

핵심적인 기능은 단축키를 통해 사용합니다. 코드를 수정하고 싶을 때는 수정할 영역을 선택한 뒤 맥의 경우 Cmd+K, 윈도우의 경우 Ctrl+K를 누르면 입력창이 나타납니다.

### 사용 예시
실제 작업 환경에서 다음과 같이 활용해 보세요.

첫 번째 활용: 코드 영역을 선택하고 Cmd+K를 누른 뒤 "이 부분에 에러가 발생했을 때 처리하는 내용을 추가해줘"라고 요청해 보세요. 인공지능이 코드를 분석하여 적절한 방어 코드를 작성해 줍니다.

두 번째 활용: 오른쪽 채팅 창을 열고 프로젝트 전체에 대해 궁금한 점을 물어보세요. "이 프로젝트가 어떻게 동작하는지 설명해줘"라고 물으면 파일들을 훑어보고 전체적인 흐름을 짚어줍니다.

세 번째 활용: 코드를 작성하다 보면 인공지능이 다음에 올 내용을 미리 예상하여 흐릿하게 보여줍니다. 이때 키보드의 Tab 키를 누르면 제안된 코드가 즉시 입력됩니다.

### 비용 안내
Cursor는 무료로 사용할 수 있는 기본 플랜과 더 많은 인공지능 모델을 제한 없이 사용할 수 있는 유료 플랜으로 나뉩니다. 입문 단계에서는 무료 플랜으로도 충분히 많은 작업을 경험해 볼 수 있습니다.

## 4. Claude Code vs Cursor — 어떤 걸 쓸까?

두 도구는 각자의 장점이 뚜렷합니다. 아래 표를 통해 비교해 보겠습니다.

| 항목 | Claude Code | Cursor |
| :--- | :--- | :--- |
| 동작 방식 | 터미널 기반 (명령어 입력 방식) | 코드 편집기 기반 (시각적 방식) |
| 설치 방법 | npm을 통한 명령어 설치 | 전용 설치 파일 다운로드 |
| 시각적 편집 | 텍스트 대화로만 진행 | 화면에서 직접 코드를 보며 수정 |
| 파일 관리 | AI가 직접 생성하고 관리 | 사용자가 직접 확인하며 수정 가능 |
| 추천 상황 | 프로젝트 초기 설정이나 빠른 실행 시 | 코드의 세부 내용을 보며 고쳐나갈 때 |

가장 좋은 방법은 두 도구를 모두 설치해 두는 것입니다. 프로젝트를 처음 시작하거나 한꺼번에 많은 파일을 설정해야 할 때는 Claude Code를 사용하고, 완성된 코드를 세밀하게 다듬거나 시각적으로 확인하며 작업할 때는 Cursor를 활용하는 식으로 상황에 맞게 골라 쓰는 것을 권장합니다.

## 5. 자주 만나는 문제와 대처법

새로운 도구를 사용하다 보면 예기치 못한 상황이 발생하곤 합니다. 대표적인 사례들을 살펴보겠습니다.

첫 번째: Claude Code 실행 시 "npm: command not found"라는 메시지가 나옵니다.
이것은 도구가 동작할 엔진인 Node.js가 설치되지 않았을 때 발생합니다. Node.js를 먼저 설치하고 다시 시도해 보세요.

두 번째: 인증 과정에서 오류가 발생하거나 화면이 넘어가지 않습니다.
브라우저에서 팝업 차단 기능이 켜져 있는지 확인해 보세요. 인증을 위한 새 창이 열리는 것을 막고 있을 수 있습니다.

세 번째: Cursor에서 무료 사용 한도를 초과했다는 안내가 나옵니다.
무료 플랜은 한 달 동안 사용할 수 있는 횟수가 정해져 있습니다. 한도가 초과되었다면 다음 달까지 기다리거나 유료 플랜으로 전환을 고려해야 합니다.

네 번째: 인공지능의 응답 속도가 너무 느립니다.
인터넷 연결 상태를 확인해 보세요. 또한 전 세계적으로 사용자가 몰리는 시간에는 서버 부하로 인해 속도가 일시적으로 느려질 수 있습니다.

다섯 번째: 인공지능이 작성해 준 코드가 제대로 동작하지 않습니다.
가장 빈번하게 발생하는 일입니다. 이때 당황하지 말고 "방금 작성한 코드가 작동하지 않아. 이전 상태로 되돌려줘"라고 요청하거나, 미리 저장해 둔 시점으로 수동 복구를 진행해야 합니다. 되돌리기 능력이 곧 실력이라는 점을 잊지 마세요.

## 6. 도구 삭제하기

만약 도구를 더 이상 사용하지 않거나 새로 설치하고 싶다면 다음과 같은 방법으로 삭제할 수 있습니다.

Claude Code를 삭제하려면 터미널에 다음 명령어를 입력합니다.
```bash
npm uninstall -g @anthropic-ai/claude-code
```

Cursor를 삭제하려면 일반적인 프로그램을 지울 때와 같은 방식을 사용합니다. 윈도우라면 제어판의 프로그램 추가/제거 기능을 이용하고, 맥이라면 응용 프로그램 폴더에서 Cursor 앱을 휴지통으로 이동시키면 됩니다.

## 7. 정리를 마치며

지금까지 Claude Code와 Cursor라는 두 가지 혁신적인 도구에 대해 알아보았습니다. Claude Code는 터미널에서 신속하게 명령을 수행하는 듬직한 일꾼이고, Cursor는 우리가 코드를 눈으로 보며 함께 고민할 수 있는 지혜로운 조력자입니다.

인공지능 코딩 도구는 우리에게 강력한 힘을 주지만, 결국 그 도구를 사용하는 주체는 여러분 자신입니다. 비서가 가져온 결과물이 올바른지 판단하고, 문제가 생겼을 때 환경을 이해하여 되돌릴 수 있는 능력을 갖추는 것이 진정한 바이브코딩의 고수가 되는 길입니다.

이 도구들을 활용해 여러분만의 멋진 아이디어를 현실로 만들어보시길 응원합니다. 코딩의 즐거움은 이제부터 시작입니다.
